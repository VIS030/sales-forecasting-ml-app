About This Project

A complete sales intelligence platform that connects data cleaning, EDA, machine learning, and Flask web delivery into one practical business workflow.

Project Overview

This system is designed like a mini analytics product used by startups and e-commerce teams. It transforms raw transactional records into actionable insights and prediction outputs.

  • Data cleaning + feature preparation pipeline
  • SQLite storage for fast KPI querying
  • Dashboard visual analytics with filters
  • ML-powered sales estimation API
Problem Statement

E-commerce teams need accurate demand forecasting and fast insights to optimize promotions, inventory, and product strategy. This solution provides visual analytics and model-based sales predictions to reduce guesswork.

Example: Before launching a campaign, managers can test price, quantity, and discount combinations on the prediction page to estimate expected revenue impact.

Dataset Description

The system uses a large transaction dataset containing user demographics, product details, purchase behavior, discounts, and review scores. Data is cleaned and stored in SQLite to power dashboard KPIs and EDA.

Core fields include Price, Quantity, DiscountRate, Category, Country, and target variable TotalAmount.

Machine Learning Models
  • Linear Regression (baseline)
  • Random Forest Regressor
  • XGBoost Regressor

Models are compared using R2 and MAE, and the best-performing model is automatically saved and loaded for live predictions.

EDA (Exploratory Data Analysis)

The EDA section helps users understand historical patterns before acting on predictions.

  • Sales trend chart reveals seasonality and growth shifts
  • Category and gender pie charts show customer mix
  • Age distribution highlights dominant buyer segments
  • Discount vs sales scatter shows promotion behavior
Flask Backend Architecture

Flask acts as the orchestration layer between database analytics, machine learning inference, and frontend templates.

  • Routes serve Home, Dashboard, Predict, EDA, and About pages
  • Session auth protects private analytics routes
  • `/api/predict` handles JSON input and returns predictions
  • SQLite queries power KPI cards and chart data dynamically
